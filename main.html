Question-1: What are the different ways to select an element in the DOM?

1.getElementById() → Select an element by ID Example:document.getElementById("myId");
2.getElementsByClassName() → Select all elements with a specific class Example:document.getElementsByClassName("myClass");
3.getElementsByTagName() → Select all elements of a specific tag Example:document.getElementsByTagName("p");
4.querySelector() → Select the first matching element using a CSS selector Example:document.querySelector(".myClass"); // Select by class document.querySelector("#myId"); // Select by ID
5.querySelectorAll() → Select all matching elements (returns a NodeList)

Question-2: What is the difference between innerHTML, innerText, and textContent ?

1.Inner HTML
Shows or changes HTML + text inside an element. Understands HTML tags.
2.Inner Text
Shows or changes only visible text. Ignores hidden text and does not read HTML tags..
3.textContent
Shows or changes all text, including hidden text. Does not read HTML tags.

Question-3: What is event delegation in the DOM?

Event delegation in JavaScript is a pattern that efficiently handles events. Events can be added to a parent element instead of adding to every single element. It refers to the process of using event propagation (bubbling) to handle events at a higher level in the DOM than the element on which the event originated.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is a process where an event starts from the target element and moves up to its parent elements. When an event occurs on a child element, it first runs the handler on that element and then triggers the same event on its ancestors.

Question-5: How do you create, add, and remove elements using JavaScript?

1.Create an Element You can create an element using document.createElement(). let newDiv = document.createElement("div"); newDiv.textContent = "Hello, I am a new div!";
2.Add an Element Use appendChild() or append() to add it inside another element. document.body.appendChild(newDiv); or document.getElementById("parent").appendChild(newDiv); // Adds inside a specific parent
3. Remove an Element Use removeChild() or remove() to delete an element. document.getElementById("parent").removeChild(newDiv); // Remove from parent
